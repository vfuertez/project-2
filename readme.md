# App Name

Bookmarker

Creates a name of a guide, tutorial or any websites that you plan to save and add details of url, creator, and notes.

# Technologies Used

* Express 
* Mongoose
* EJS
* Javascript
* CSS 
* Node
* MongoDB
* Render
